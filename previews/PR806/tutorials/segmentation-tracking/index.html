Segmentation and Tracking

This notebook demonstrates a standard segmentation and tracking workflow. It loads a vector of example images, segments them, then runs a tracking algorithm to match floes in the images.

In this example, we use two images. However, the same syntax can be used for a larger number of images.

Set up the environment.


using Pkg
Pkg.add(;name="IceFloeTracker", rev="806/merge")
Pkg.add("Images")

Load packages and functions required. 

using IceFloeTracker: LopezAcosta2019Tiling, Watkins2026Dataset, FloeTracker, FilterFunction, MinimumWeightMatchingFunction, modis_truecolor, modis_falsecolor, modis_landmask, info
using Images: erode, segment_mean, labels_map, SegmentedImage, RGB, mosaicview
using Dates: DateTime

Run the tracker on the segmentation results. The tracker takes a vector of segmented images and requires the observation time of each for filtering candidate matches.

tracker = FloeTracker(;
    filter_function=FilterFunction(),
    matching_function=MinimumWeightMatchingFunction()
)
tracking_results = tracker(segmentation_results, info(dataset).pass_time)
